# tftp_common

## Dependencies

| Package name            | Minimum required version | Ubuntu 22.04                         |
|-------------------------|--------------------------|--------------------------------------|
| C++                     | C++11                    | sudo apt-get install build-essential |
| Boost (Spirit, Phoenix) | 1.74                     | sudo apt-get install libboost-dev    |

## Installation instructions

```bash
mkdir build
cd build/
cmake ..
make install
```
